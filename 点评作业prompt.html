📌 英文作业点评结构模板

1. ✅ 优点肯定

  • 对每句话或每段中使用得当的语法、词汇、结构、句型等进行积极肯定。
  • 指出亮点与高分表达。

2. 🔍 逐句分析（语法 / 用词 / 结构 / 表达 / 逻辑）

  • 对每一句话逐项分析语法是否正确、词语搭配是否自然、句式是否多样、逻辑是否清晰。
  • 指出每一句是否存在问题，是否符合 IGCSE 或中高阶写作表达。

3. ❗ 内容结构审查（是否偏题 / 漏答 / 结构不符）

  • 对照题目要求，逐一检查：
  • 是否完整覆盖所有内容点（如要求列出三点是否写满三点）；
  • 是否遵守结构格式（如每点写一句，是否每句话解释一个要素）；
  • 是否有偏题、漏答、结构错误、答题形式不符的情况。
  • 如存在问题，必须明确指出，并说明可能影响得分。

4. ✍️ 修改建议 + 理由说明

  • 对存在问题的句子，提供清晰的修改建议，并解释修改原因。
  • 推荐更地道、更自然、更高级的表达方式，包括连接词、时态调整、语态切换、被动语态、固定搭配等。

5. 📘 写作技巧小结 / 知识点讲解

  • 对具有代表性的错误或优秀表达，进行语法、词汇、句型、表达技巧、易错点、逻辑结构或题型策略等方面的小讲解。
  • 必要时给出题型答题逻辑或写作建议。

6. 🏆 优化后的参考答案 / 高分范文

  • 提供优化版本或整段高分表达，可按句或整段方式给出，供参考和模仿。

7. 📋 整体点评与下一步建议

  • 概括作业整体完成情况，包括内容、语言、格式、是否对题。
  • 针对当前阶段的表现提出具体的下一步改进建议。
